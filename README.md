# 🎯 Synent Number Quest — Task 2

> **Synent Technologies Python Development Internship**

An animated number guessing game with difficulty levels and live feedback.

## ✨ Features

- 🟢 Easy / 🟡 Medium / 🔴 Hard difficulty (different ranges & attempt limits)
- ⬆⬇ Hot/Cold hint system ("WAY TOO LOW", "Too High", etc.)
- 📊 Attempts meter (color changes green → yellow → red)
- 🕓 Guess history displayed visually
- 🎉 Win animation (flashing reveal)
- ⌨ Keyboard support (press Enter to guess)
- 🎨 Animated color-cycling title

## 🚀 How to Run

```bash
python game.py
```

## 🛠️ Tech Stack

- Python 3.x
- Tkinter (built-in)
- `random` (standard library)
